    merger(arg1, arg2, group_tail)

    View Source

    This function will merge an itemset with another itemset.

    What is merge itemsets and make sub itemset? if we have a = [1, 2, 4] and b = [1, 2, 5] then merge of them will be: result = [1, 2, 4, 5]

    a and b can merge because of (0 .. k-1)th items in their lists are similar.

    In this module for avoiding of list overhead, we merge lists by (1 .. k)th items. if a = [2, 1] and b = [3, 1] then merge of them with this algorithm will be: result = [3, 2, 1]

    Examples 

    iex> DataMiner.Eclat.merge_itemsets([{[2, 1], MapSet.new([2])}, {[3, 1], MapSet.new([2])}])
[{[3, 2, 1], #MapSet<[2]>}]

    remove_low_frequencies(itemsets, min_supp, transactions_length)

    View Source

    When itemsets merged succesfully, we should pipe them into remove_low_frequencies that will remove all of itemsets that size of their transactions are lower that minimum support.

    This is for downward closers!

    Examples 

    iex> DataMiner.Eclat.remove_low_frequencies([{[:a], MapSet.new([1, 2, 3])}, {[:b], MapSet.new([1])}], 50, 3)
[{[:a], #MapSet<[1, 2, 3]>}]

    transactios_to_eclat_form(transactions)

    View Source

    This function will get main transactions and return eclat form of that.

    Eclat form is a map that show transactions of an item that it is inside them!

    Examples 

    iex> DataMiner.Eclat.transactios_to_eclat_form([["1", "2"], ["2", "4"], ["1", "5"], ["1", "6", "7", "3", "1", "2", "9"]])
{[:"1"] => #MapSet<[0, 2, 3]>, [:"2"] => #MapSet<[0, 1, 3]>, [:"3"] => #MapSet<[3]>, [:"4"] => #MapSet<[1]>, [:"5"] => #MapSet<[2]>, [:"6"] => #MapSet<[3]>, [:"7"] => #MapSet<[3]>, [:"9"] => #MapSet<[3]>}
